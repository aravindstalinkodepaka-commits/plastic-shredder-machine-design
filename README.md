# Plastic Shredder Machine Design

## Project Overview

This project focuses on the design and development of a **Plastic Shredder Machine** for reducing plastic waste into smaller pieces, making it easier to handle and process for recycling applications.

The project was developed as part of a **Machine Drawing course**, with emphasis on engineering drawing principles, component representation, assembly design, dimensional accuracy, and practical mechanical design.

## Objectives

* Design and represent a plastic shredder machine using engineering drawing principles.
* Develop detailed drawings of individual machine components.
* Create a complete assembly drawing of the machine.
* Apply standard dimensioning and machine drawing conventions.
* Understand the arrangement and interaction of mechanical components.
* Explore the application of mechanical design in plastic waste processing.

## Working Principle

The plastic shredder machine operates by feeding plastic waste through a hopper into the shredding chamber. The cutting mechanism applies shear and cutting forces to break the plastic material into smaller pieces.

The shredded plastic is then discharged from the machine for further handling, recycling, or processing.

## Main Components

The major components of the machine include:
